# MOst important Git Command With the usage

  | Command                         | Usage / Meaning                            |
| ------------------------------- | ------------------------------------------ |
| `git --version`                 | Check installed Git version                |
| `git init`                      | Start Git in a project folder              |
| `git clone <url>`               | Download repository from GitHub            |
| `git status`                    | Show changed/untracked files               |
| `git add filename`              | Add one file to staging area               |
| `git add .`                     | Add all changed files                      |
| `git commit -m "msg"`           | Save a version/snapshot of code            |
| `git log`                       | Show commit history                        |
| `git log --oneline`             | Show short commit history                  |
| `git diff`                      | Show code changes before commit            |
| `git restore filename`          | Undo changes in a file                     |
| `git restore --staged filename` | Remove file from staging area              |
| `git reset --hard`              | Delete all local changes and reset project |
| `git remote add origin <url>`   | Connect local project to GitHub repo       |
| `git remote -v`                 | Show connected GitHub repository           |
| `git push`                      | Upload code to GitHub                      |
| `git push origin main`          | Push code to main branch                   |
| `git push -u origin main`       | First-time push and link branch            |
| `git pull`                      | Download latest code and merge             |
| `git fetch`                     | Download updates without merging           |
| `git branch`                    | Show all branches                          |
| `git branch branch-name`        | Create new branch                          |
| `git checkout branch-name`      | Switch branch                              |
| `git checkout -b branch-name`   | Create + switch branch                     |
| `git switch branch-name`        | Modern way to switch branch                |
| `git merge branch-name`         | Combine another branch into current branch |
| `git stash`                     | Temporarily save uncommitted changes       |
| `git stash pop`                 | Restore stashed changes                    |
| `git stash list`                | Show saved stashes                         |
| `git clean -f`                  | Delete untracked files                     |
| `git tag v1.0`                  | Create project version tag                 |
| `git tag`                       | Show all tags                              |
